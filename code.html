omas.omas_core.aggregate_imas_html_docs(imas_html_dir=u'/Users/meneghini/tmp/imas', imas_version=u'3.10.1')[source]¶

this function aggregates all of the IMAS html documentation pages into a single clean.html page that is stored under the imas_structures folder.

omas/imas_structures/<imas_version>/clean.html

This clean.html page is then –MANUALLY–: 1. opened in EXCEL 2. in EXCEL all cells are un-merged 3. the EXCEL document is finally saved as clean.xls

omas/imas_structures/<imas_version>/clean.xls
Parameters:
  • imas_html_dir – directory where the IMAS html documentation is stored
  • imas_version – IMAS version
